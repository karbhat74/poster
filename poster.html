Social Media Wellness Analyzer and
Recommendation System for Digital Well-being

Guide: Dr. Rekha P M | Team: Karthik A S, Anusha Karnam, Badri H S, Adarsh Naik

ABSTRACT

This project introduces an AI-driven system that analyzes social media usage and predicts mental wellness levels using machine learning and NLP. It captures emotional sentiment, behavioral trends, usage patterns, and linguistic cues to estimate digital well-being.

PROBLEM STATEMENT

Excessive social media use increases stress, anxiety, sleep loss, and digital fatigue. Existing tools track only screen-time and fail to detect emotional or cognitive distress.

KEY FEATURES

  • AI-based mental wellness prediction
  • NLP sentiment analysis
  • Behavioral pattern clustering
  • Personalized recommendations
  • Visual analytics dashboard
  • Early distress detection

SYSTEM ARCHITECTURE

IMPLEMENTATION

Languages: Python, HTML, CSS, JS

Libraries: Scikit-Learn, Pandas, NumPy

Visualization: Matplotlib, Seaborn

Models: Random Forest, Logistic Regression, SVM, K-Means

RESULTS

• Accurate prediction of mental wellness scores • Clustering reveals Low-Risk, Medium-Risk, High-Risk groups • Sentiment scores strongly correlate with stress

CONCLUSION

The system analyzes digital behavior, predicts early mental-health risks, and provides wellness recommendations. Future enhancements include real-time monitoring and multimodal emotion detection.
